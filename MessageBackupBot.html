Introduction to MessageBackupBot

The MessageBackupBot, is the all-new discord bot, allowing guild messages to be backed up and restored. In this documentary, we will show you how to set it up and how to use it.

Legal Documentation

https://docs.snaildos.com/en/Message-Backup-Bot-Legal

Installation

To start using the bot, you need to invite it by clicking here.

Then, you want to select the “invite” link, and press “Invite To Your Server”. Please make sure you give it administrator or at least these permissions:

  • Manage Webhooks
  • Manage Emojis and Stickers
  • Send Messages
  • Read Messages
  • Manage Server
  • Manage Channels
  • Embed Links
  • Attach Files
  • Slash Commands

Without these permissions, the bot will fail to work as expected.

Getting Started

After you added the bot to your discord server, please type in /user register. Then, confirm the agreement. 

Once this is complete, you can run /backups create to create a new backup.

At any time, feel free to use /user language to change your language.

You can use /backups load to load your backup as well by supplying your encryption key.

Use the auto-fill discord feature to select your backups, to view and delete, type in /backups delete
