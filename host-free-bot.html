Official Instructions!

  1. Join our discord, this won't work without it. https://invite.gg/snaildos
  2. Head to our panel site, https://server.snaildos.com
  3. Select register, enter an email, username and password.
  4. Press signup, then press login.
  5. Once done, head to ‘server’ and buy the free server.
  6. You can select which type of serevr you want such as; discord.py or discord.js
  7. Once done, press create.
  8. Click manage to head to our main panel to configure.
  9. Please wait a while while the server set's up.

How to install a Discord Bot on this service?

  1. You should be familiar with Git, if not, you will have a hard time.
  2. On your git repo address, type in your GitHub Repo URL, install branch can/should always be master. Leave User uploaded files and Auo Update to default. On your Git USERNAME, type your username.
  3. For the service to function properly, type your github acess token. For github: https://github.com/settings/tokens type that URL, press “generate” and then copy and paste it in. No documentary exists for GitLab currently.
  4. After that, return to console and hit “start”. In a matter of minutes, your bot will start. Give your bot about 10-20 mins to load up for the first time.
